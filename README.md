 Student Performance Analysis

 Overview
This project explores the factors that influence student academic performance using Python.
The goal is to identify patterns and insights that can help educators and learners improve outcomes.

 Dataset
The dataset contains student-related attributes such as:
- Gender
- Parental education level
- Test preparation course
- Math, reading, and writing scores

 Tools Used
- Python
- pandas
- matplotlib
- Jupyter Notebook

 Key Insights
- Students who completed test preparation courses performed better on average.
- Parental education level shows a noticeable impact on student scores.
- Reading and writing scores are strongly correlated.

Why This Project Matters
As a mathematics teacher, this analysis connects data science with real educational insights,
showing how data can support better learning decisions.
